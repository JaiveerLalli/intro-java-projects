# Intro Java Projects

This repository contains several Java projects completed as part of the **CSE 121–123** course series.  
Each project focuses on applying fundamental programming concepts such as methods, loops, arrays, and object-oriented design.

## Projects

### [Patient Prioritizer](./Patient-Prioritizer)
A program that determines patient care priority based on various health indicators.  
Implements scoring methods to simulate medical triage systems.

### [Password Protector](./Password-Protector)
A password security simulator that can generate complex passwords, verify login attempts, and brute-force guess a stored password.  
Includes random password generation and timing functionality.

### [Election Simulator](./Election-Simulator)
A simulation that models how votes are counted across multiple candidates and districts.  
Uses arrays, loops, and conditional logic to calculate winners and display results.

---

Each folder contains:
- `ProjectName.java` — main source file  
- `README.md` — explanation of the project, its purpose, and how it works
